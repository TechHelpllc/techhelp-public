MyTechHelp Support

MyTechHelp helps users receive remote assistance from trusted helpers through guided support sessions, screen sharing, and optional FaceTime Audio calling.

Need help? Contact MyTechHelp support at MyTechHelpSolutionsLLC@outlook.com.

Getting Started

To request help, open MyTechHelp and choose the option to request support. You may be able to connect with a trusted helper or provide a help code to someone assisting you.

Screen Sharing

The requester controls screen sharing. Helpers can only view the requester’s screen while sharing is active. Helpers cannot take control of the requester’s device.

FaceTime Audio Calls

MyTechHelp may help start a FaceTime Audio call between the requester and helper. FaceTime calls are handled by Apple FaceTime and are not controlled, recorded, or stored by MyTechHelp.

Ending a MyTechHelp session does not automatically end a FaceTime call. Please end the FaceTime call separately when finished.

Troubleshooting

  • Make sure both users are connected to the internet.
  • Make sure FaceTime is enabled if using the call feature.
  • Make sure screen sharing is started by the requester.
  • If notifications are not working, check notification permissions for MyTechHelp.
  • If a session does not connect, close and reopen the app and try again.
